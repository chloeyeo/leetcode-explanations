#455 Assign Cookies Easy by LeetCode
Algorithm Visualization
Sort both arrays. Use smallest cookie that satisfies each child.

g = [1, 2, 3]   (greed, sorted)
s = [1, 1, 2]   (cookies, sorted)

Step 1: s[0]=1 >= g[0]=1  KEEP  keep_end=2  content=1
Step 2: s[1]=1 <  g[1]=2  SKIP
Step 3: s[2]=2 >= g[1]=2  KEEP  content=2

Result: 2 children satisfied
Basic Solution
def findContentChildren(self, g, s): g.sort(); s.sort() child = cookie = 0 while child < len(g) and cookie < len(s): if s[cookie] >= g[child]: child += 1 cookie += 1 return child
Optimized Solution
Pythonic for-loop
def findContentChildren(self, g, s): g.sort(); s.sort() gi = 0 for c in s: if gi < len(g) and c >= g[gi]: gi += 1 return gi

#435 Non-overlapping Intervals Medium by LeetCode
Algorithm Visualization
Sort by END time. Keep intervals ending earliest.

[[1,2],[2,3],[3,4],[1,3]] -> sorted by end -> [[1,2],[2,3],[1,3],[3,4]]

[1,2]: start=1 >= -inf  KEEP  prev_end=2
[2,3]: start=2 >= 2     KEEP  prev_end=3
[1,3]: start=1 <  3     REMOVE  count=1
[3,4]: start=3 >= 3     KEEP  prev_end=4

Answer: 1 removal
Basic Solution
def eraseOverlapIntervals(self, intervals): if not intervals: return 0 intervals.sort(key=lambda x: x[1]) remove, prev_end = 0, float('-inf') for s, e in intervals: if s >= prev_end: prev_end = e else: remove += 1 return remove
Optimized Solution
Count kept, return n - kept
def eraseOverlapIntervals(self, intervals): intervals.sort(key=lambda x: x[1]) kept, prev_end = 0, float('-inf') for s, e in intervals: if s >= prev_end: kept += 1; prev_end = e return len(intervals) - kept
Complexity
ComponentTimeSpaceReason

#135 Candy Hard by LeetCode
Algorithm Visualization
Two-pass greedy: left-to-right then right-to-left

ratings = [1, 0, 2]
Init:    candies = [1, 1, 1]

Pass 1 (L->R, enforce higher rating than left neighbor gets more):
  i=1: 0 < 1  no change   candies=[1,1,1]
  i=2: 2 > 0  +1          candies=[1,1,2]

Pass 2 (R->L, enforce higher rating than right neighbor gets more):
  i=1: 0 < 2  no change
  i=0: 1 > 0  max(1,1+1)=2  candies=[2,1,2]

Total = 2+1+2 = 5
Basic Solution (Two-Pass)
def candy(self, ratings): n = len(ratings) c = [1] * n for i in range(1, n): if ratings[i] > ratings[i-1]: c[i] = c[i-1] + 1 for i in range(n-2, -1, -1): if ratings[i] > ratings[i+1]: c[i] = max(c[i], c[i+1] + 1) return sum(c)
Optimized Solution (O(1) space)
Slope counting — no extra array
def candy(self, ratings): total, up, down, peak = 1, 0, 0, 0 for i in range(1, len(ratings)): d = ratings[i] - ratings[i-1] if d > 0: up += 1; down = 0; peak = up total += up + 1 elif d < 0: down += 1; up = 0 total += down + (1 if down >= peak else 0) else: up = down = peak = 0; total += 1 return total
