💡 Basic Solution
class Solution: def findContentChildren(self, g: list[int], s: list[int]) -> int: # Sort both arrays to use greedy matching g.sort() s.sort() child, cookie = 0, 0 while child < len(g) and cookie < len(s): if s[cookie] >= g[child]: child += 1 # satisfy this child cookie += 1 # move to next cookie return child
⚡ Optimized Solution
class Solution: def findContentChildren(self, g: list[int], s: list[int]) -> int: g.sort(); s.sort() j = 0 # Iterate through cookies; greedily assign smallest sufficient for cookie in s: if j < len(g) and cookie >= g[j]: j += 1 return j

LeetCode #45 — Jump Game II
by LeetCode
Medium
📊 Algorithm Visualization
nums = [2, 3, 1, 1, 4] Jump 1: start at idx 0 (val=2), can reach idx 1 or 2 farthest = max(0+2)=2 [0]------>[[1][2] Jump 2: from range [1,2], best farthest: idx1 val3 -> 1+3=4 idx2 val1 -> 2+1=3 farthest = 4 [1][2]-------->[[3][4] Reached end (idx 4)! jumps = 2 true
💡 Basic Solution
class Solution: def jump(self, nums: list[int]) -> int: jumps = current_end = farthest = 0 for i in range(len(nums) - 1): farthest = max(farthest, i + nums[i]) if i == current_end: jumps += 1 current_end = farthest if current_end >= len(nums) - 1: break return jumps
⚡ Optimized Solution
class Solution: def jump(self, nums: list[int]) -> int: n = len(nums) jumps = cur_end = far = 0 for i in range(n - 1): far = max(far, i + nums[i]) if i == cur_end: jumps += 1; cur_end = far return jumps
📈 Complexity Analysis
VersionTimeSpaceReason
BasicO(n)O(1)Single pass; early break when end reached
OptimizedO(n!O(1)Same; cleaner code without early break needed
LeetCode #135 — Candy
by LeetCode
Hard
📊 Algorithm Visualization
ratings = [1, 0, 2] Pass 1 (left -> right): idx0: base=1 candies=[1,1,1] idx1: 0 < 1, keep=1 candies=[1,1,1] idx2: 2 > 0, +1 candies=[1,1,2] Pass 2 (right -> left): idx1: 0 < 2, check candy[1]=max(1,candy[2]+1)=max(1,3)=3 candies=[1,3,2] idx0: 1 > 0, check candy[0]=max(1,candy[1]+1)=max(1,4)=4 candies=[4,3,2] (ratings=[1,0,2] result: [1,1,2] -> total=4)
💡 Basic Solution
class Solution: def candy(self, ratings: list[int]) -> int: n = len(ratings) candies = [1] * n for i in range(1, n): if ratings[i] > ratings[i - 1]: candies[i] = candies[i - 1] + 1 for i in range(n - 2, -1, -1): if ratings[i] > ratings[i + 1]: candies[i] = max(candies[i], candies[i + 1] + 1) return sum(candies)
⚡ Optimized Solution (O(1) space)
class Solution: def candy(self, ratings: list[int]) -> int: n, total, up, down, peak = len(ratings), 1, 0, 0, 0 for i in range(1, n): if ratings[i] > ratings[i-1]: up += 1; down = 0; peak = up total += up + 1 elif ratings[i] < ratings[i-1]: down += 1; up = 0 total += down + (1 if peak >= down else 2) else: up = down = peak = 0; total += 1 return total
📈 Complexity Analysis
VersionTimeSpaceReason
BasicO(n)O(n)Two passes + candies array of size n
OptimizedO(n)O(1)Single pass tracking up/down slopes; no extra array
