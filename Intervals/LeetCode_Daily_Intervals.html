Meeting Rooms Easy

© LeetCode — for educational purposes
📐 알고리즘 시각화
intervals = [[0,30],[5,10],[15,20]]

Sort by start: [0,30] [5,10] [15,20]
Check: [0,30] vs [5,10] → 5 < 30 ❌ OVERLAP → return False

No overlap: [[7,10],[2,4]] → sort → [[2,4],[7,10]]
  [2,4] vs [7,10] → 7 >= 4 ✅ → return True

Timeline:
  0────────────────────30
       5────10
              15────20
  ↑ overlapping!
💡 기본 풀이 코드
Basic — Sort + Linear Scan
class Solution:
    def canAttendMeetings(self, intervals):
        intervals.sort(key=lambda x: x[0])
        for i in range(1, len(intervals)):
            if intervals[i][0] < intervals[i-1][1]:
                return False
        return True
⚡ 최적화 풀이
Optimized — Early Exit with any()
class Solution:
    def canAttendMeetings(self, intervals):
        intervals.sort()
        return not any(intervals[i][0] < intervals[i-1][1]
                       for i in range(1, len(intervals)))

Merge Intervals Medium

© LeetCode — for educational purposes
📐 알고리즘 시각화
Input: [[1,3],[2,6],[8,10],[15,18]]
After sort: [[1,3],[2,6],[8,10],[15,18]]
merged = [[1,3]]

curr=[2,6]: 2<=3 → merge → [1,max(3,6)]=[1,6]
curr=[8,10]: 8>6 → append [8,10]
curr=[15,18]: 15>10 → append [15,18]

Result: [[1,6],[8,10],[15,18]] ✅

  1──────6   8────10   15────18
  (merged)   (kept)    (kept)
💡 기본 풀이 코드
Basic — Sort + Merge
class Solution:
    def merge(self, intervals):
        intervals.sort(key=lambda x: x[0])
        merged = [intervals[0]]
        for start, end in intervals[1:]:
            if start <= merged[-1][1]:
                merged[-1][1] = max(merged[-1][1], end)
            else:
                merged.append([start, end])
        return merged
⚡ 최적화 풀이
Optimized — In-place tracking
class Solution:
    def merge(self, intervals):
        intervals.sort()
        res = []
        cs, ce = intervals[0]
        for s, e in intervals[1:]:
            if s <= ce: ce = max(ce, e)
            else: res.append([cs, ce]); cs, ce = s, e
        res.append([cs, ce])
        return res

Insert Interval Hard

© LeetCode — for educational purposes
📐 알고리즘 시각화
intervals=[[1,3],[6,9]], newInterval=[2,5]

Phase 1: end < newStart → [1,3]: 3<2? No → skip
Phase 2: overlaps w/new:
  [1,3]: 1<=5 and 2<=3 → YES → merged=[min(1,2),max(3,5)]=[1,5]
  [6,9]: 6<=5? NO → stop
Phase 3: append [6,9]

Result: [[1,5],[6,9]] ✅

Before:  1───3     6────9
New:       2─────5
After:  1──────5   6────9
💡 기본 풀이 코드
Basic — 3-Phase Linear Scan
class Solution:
    def insert(self, intervals, newInterval):
        res = []; i = 0; n = len(intervals)
        while i < n and intervals[i][1] < newInterval[0]:
            res.append(intervals[i]); i += 1
        while i < n and intervals[i][0] <= newInterval[1]:
            newInterval[0] = min(newInterval[0], intervals[i][0])
            newInterval[1] = max(newInterval[1], intervals[i][1])
            i += 1
        res.append(newInterval)
        return res + intervals[i:]
⚡ 최적화 풀이
Optimized — Binary Search
import bisect
class Solution:
    def insert(self, intervals, newInterval):
        starts = [iv[0] for iv in intervals]
        ends   = [iv[1] for iv in intervals]
        lo = bisect.bisect_left(ends, newInterval[0])
        hi = bisect.bisect_right(starts, newInterval[1])
        ms = min(newInterval[0], intervals[lo][0]) if lo < hi else newInterval[0]
        me = max(newInterval[1], intervals[hi-1][1]) if lo < hi else newInterval[1]
        return intervals[:lo] + [[ms, me]] + intervals[hi:]
