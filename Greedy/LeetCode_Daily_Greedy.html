LeetCode #455 - Assign Cookies Easy by LeetCode

Basic Solution (Python)
class Solution: def findContentChildren(self, g, s): g.sort(); s.sort() child = cookie = 0 while child < len(g) and cookie < len(s): if s[cookie] >= g[child]: child += 1 cookie += 1 return child
Optimized (bisect)
import bisect class Solution: def findContentChildren(self, g, s): g.sort(); s.sort() count = used = 0 for greed in g: idx = bisect.bisect_left(s, greed, used) if idx < len(s): count += 1; used = idx+1 else: break return count

LeetCode #435 - Non-overlapping Intervals Medium by LeetCode
ASCII Algorithm Visualization
intervals = [[1,2],[2,3],[3,4],[1,3]] sort by end: [[1,2],[2,3],[1,3],[3,4]] [1,2] keep end=2 [2,3] keep end=3 [1,3] OVERLAP (start=1 < end=3) remove! count=1 [[3,4] keep end=4 Removals: 1
Basic Solution
class Solution: def eraseOverlapIntervals(self, intervals): if not intervals: return 0 intervals.sort(key=lambda x: x[1]) end, remove = intervals[0][1], 0 for s, e in intervals[1:]: if s < end: remove += 1 else: end = e return remove
Optimized
class Solution: def eraseOverlapIntervals(self, intervals): intervals.sort(key=lambda x: x[1]) kept, end = 1, intervals[0][1] for i in range(1, len(intervals)): if intervals[i][0] >= end: kept += 1; end = intervals[i][1] return len(intervals) - kept
Complexity Analysis
CaseTimeSpaceReason
BothO(n log n)O(1)Sort dominates, O(n) scan
LeetCode #45 - Jump Game II Hard by LeetCode
ASCII Algorithm Visualization
nums = [2,3,1,1,4] Jump 0: reach = 0+2 = 2 currEnd=0 --> jump+1, currEnd=2 Jump 1: max(reach, 0+2, 1+3, 2+1) = 4 currEnd=2 --> jump+1, currEnd=4 currEnd=4 >= len-1, done! jumps=2
Basic Solution
class Solution: def jump(self, nums): jumps = curr_end = reach = 0 for i in range(len(nums) - 1): reach = max(reach, i + nums[i]) if i == curr_end: jumps += 1; curr_end = reach if curr_end >= len(nums) - 1: break return jumps
Optimized (1-liner)
class Solution: def jump(self, nums): jumps = curr_end = reach = 0 for i, v in enumerate(nums[:-1]): reach = max(reach, i + v) if i == curr_end: jumps += 1; curr_end = reach return jumps
Complexity Analysis
CaseTimeSpaceReason
BothO(n)O(1)Single pass, 3 variables only
