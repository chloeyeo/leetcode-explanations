#104. Maximum Depth of Binary Tree Easy

ASCII Visualization

3 / \ 9 20 / \ 15 7 Depth: 3

Basic Solution (DFS)

class Solution:
    def maxDepth(self, root):
        if not root:
            return 0
        return 1 + max(self.maxDepth(root.left), self.maxDepth(root.right))

Optimized (BFS)

from collections import deque
class Solution:
    def maxDepth(self, root):
        if not root: return 0
        queue, depth = deque([root]), 0
        while queue:
            depth += 1
            for _ in range(len(queue)):
                node = queue.popleft()
                if node.left: queue.append(node.left)
                if node.right: queue.append(node.right)
        return depth

Complexity Analysis

ApproachTimeSpaceReason
DFS (recursive)O(N)O(H)Visits each node once. Stack depth = tree height H.
BFS (iterative)O(N)O(W)Each node enqueued/dequeued once. Max queue size = level width W.
#236. Lowest Common Ancestor of a Binary Tree Medium

ASCII Visualization

3 / \ 5 1 / \ / \ 6 2 0 8 LCA(5, 1) = 3

Basic Solution (DFS)

class Solution:
    def lowestCommonAncestor(self, root, p, q):
        # Base case
        if not root or root == p or root == q:
            return root
        left = self.lowestCommonAncestor(root.left, p, q)
        right = self.lowestCommonAncestor(root.right, p, q)
        if left and right:
            return root
        return left if left else right

Optimized (Iterative parent map)

class Solution:
    def lowestCommonAncestor(self, root, p, q):
        parent = {root: None}
        stack = [root]
        while p not in parent or q not in parent:
            node = stack.pop()
            if node.left:
                parent[node.left] = node; stack.append(node.left)
            if node.right:
                parent[node.right] = node; stack.append(node.right)
        ancestors = set()
        while p:
            ancestors.add(p); p = parent[p]
        while q not in ancestors:
            q = parent[q]
        return q

Complexity Analysis

ApproachTimeSpaceReason
DFS (recursive)O(N)O(H)Worst case visits all nodes; call stack = height H.
IterativeO(N)O(N)Parent dict stores up to N entries.
#124. Binary Tree Maximum Path Sum Hard

ASCII Visualization

-10 / \ 9 20 / \ 15 7 Max Path: 15 -> 20 -> 7 = 42

Basic Solution (Post-order DFS)

class Solution:
    def maxPathSum(self, root):
        self.max_sum = float('-inf')
        def dfs(node):
            if not node: return 0
            # Ignore negative contributions
            left_max = max(dfs(node.left), 0)
            right_max = max(dfs(node.right), 0)
            self.max_sum = max(self.max_sum, node.val + left_max + right_max)
            return node.val + max(left_max, right_max)
        dfs(root)
        return self.max_sum

Optimized (list as nonlocal)

class Solution:
    def maxPathSum(self, root):
        res = [float('-inf')]
        def dfs(node):
            if not node: return 0
            l = max(dfs(node.left), 0)
            r = max(dfs(node.right), 0)
            res[0] = max(res[0], node.val + l + r)
            return node.val + max(l, r)
        dfs(root)
        return res[0]

Complexity Analysis

ApproachTimeSpaceReason
DFS post-orderO(N)O(H)Single pass visiting all N nodes. Stack depth = height H (O(N) worst, O(logN) balanced).
