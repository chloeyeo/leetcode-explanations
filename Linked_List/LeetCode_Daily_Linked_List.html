Easy #206 Reverse Linked List — by LeetCode
📐 Algorithm Visualization (Iterative Reversal)
Input: 1 → 2 → 3 → 4 → 5 → None prev=None curr=1 Step 1: nxt=2, 1←None, prev=1, curr=2 Step 2: nxt=3, 2←1, prev=2, curr=3 Step 3: nxt=4, 3←2, prev=3, curr=4 Step 4: nxt=5, 4←3, prev=4, curr=5 Step 5: nxt=None, 5←4, prev=5, curr=None Output: 5 → 4 → 3 → 2 → 1 → None ↑ new head = prev
🐍 Basic Solution (Python)
class Solution: def reverseList(self, head: ListNode) -> ListNode: # 재귀 방식: 끝까지 내려간 뒤 역찅 연결 if not head or not head.next: return head new_head = self.reverseList(head.next) head.next.next = head # 다음 노드가 현재를 가리키게 head.next = None # 현재 노드 포인터 끊기 return new_head
⚡ Optimized Solution (Iterative)
class Solution: def reverseList(self, head: ListNode) -> ListNode: prev, curr = None, head while curr: nxt = curr.next # 다음 저장 curr.next = prev # 포인터 역전 prev = curr # prev 전진 curr = nxt # curr 전진 return prev # 새로운 헤드

Medium #143 Reorder List — by LeetCode
📐 Algorithm Visualization (3-Step Strategy)
Input: 1 → 2 → 3 → 4 → 5 Step 1: Find middle with slow/fast pointer slow=3 (middle) Split: L1=[1→2→3] L2=[4→5] Step 2: Reverse second half L2 reversed = [5→4] Step 3: Interleave L1 and L2 1 → 5 → 2 → 4 → 3 L1: 1 → 2 → 3 L2: 5 → 4 merge: take 1 from L1, 1 from L2 alternately
🐍 Basic Solution (Python)
class Solution: def reorderList(self, head: ListNode) -> None: # 배열로 변환 후 인덱스로 재연결 nodes = [] curr = head while curr: nodes.append(curr) curr = curr.next lo, hi = 0, len(nodes) - 1 while lo < hi: nodes[lo].next = nodes[hi] lo += 1 if lo == hi: break nodes[hi].next = nodes[lo] hi -= 1 nodes[lo].next = None
⚡ Optimized Solution (O(1) Space)
class Solution: def reorderList(self, head: ListNode) -> None: # 1. 중간 찾기 slow = fast = head while fast and fast.next: slow = slow.next fast = fast.next.next # 2. 후반부 역전 prev, curr = None, slow.next slow.next = None # 전반부 끊기 while curr: nxt = curr.next curr.next = prev prev = curr curr = nxt # 3. 인터리브 합병 first, second = head, prev while second: tmp1, tmp2 = first.next, second.next first.next = second second.next = tmp1 first, second = tmp1, tmp2

Hard #23 Merge k Sorted Lists — by LeetCode
📐 Algorithm Visualization (Min-Heap)
lists = [1→4→5, 1→3→4, 2→6] Min-Heap (priority queue) approach: Init heap with head of each list: heap = [(1, list0), (1, list1), (2, list2)] Step 1: pop (1,list0) → output: 1, push list0.next=(4) heap = [(1,list1), (2,list2), (4,list0)] Step 2: pop (1,list1) → output: 1, push (3) heap = [(2,list2), (3,list1), (4,list0)] Step 3: pop (2,list2) → output: 2, push (6) heap = [(3,list1), (4,list0), (6,list2)] ...continues until heap empty Final: 1 → 1 → 2 → 3 → 4 → 4 → 5 → 6 Divide & Conquer alternative: Round 1: merge(L0,L1), merge(L2,_) = 2 lists Round 2: merge those 2 = final answer Total: O(n log k)
🐍 Basic Solution (Brute Force)
class Solution: def mergeKLists(self, lists): # 모든 값 수집 후 정렬 후 재연결 vals = [] for node in lists: while node: vals.append(node.val) node = node.next vals.sort() dummy = curr = ListNode(0) for v in vals: curr.next = ListNode(v) curr = curr.next return dummy.next
⚡ Optimized Solution (Min-Heap)
import heapq class Solution: def mergeKLists(self, lists): heap = [] for i, node in enumerate(lists): if node: heapq.heappush(heap, (node.val, i, node)) dummy = curr = ListNode(0) while heap: val, i, node = heapq.heappop(heap) curr.next = node curr = curr.next if node.next: heapq.heappush(heap, (node.next.val, i, node.next)) return dummy.next
