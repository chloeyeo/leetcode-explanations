Kth Largest Element in a Stream Easy

© LeetCode — for educational purposes
📐 알고리즘 시각화 (Min-Heap of size k)
Stream: [4, 5, 8, 2], k=3   → 3번째로 큰 값을 항상 추적

Step 1: heap = [4]           add(4)
Step 2: heap = [4,5]         add(5)
Step 3: heap = [4,5,8]       add(8) → size==k, top=4
Step 4: heap = [5,8]         add(2) → 2 < top(4), skip → still top=4
         heap = [4,5,8]      ← size-k min-heap

add(3):  3 < heap[0]=4 → skip
add(10): 10 > heap[0]=4 → push 10, pop 4
         heap = [5,8,10] → top = 5  ✅ kth largest = 5

  Min-Heap (size k):
  ┌───────────────┐
  │  heap[0]=MIN  │  ← always the kth largest
  │  heap[1]      │
  │  heap[2]=MAX  │
  └───────────────┘
