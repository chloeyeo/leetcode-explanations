#268 Missing Number Easy by LeetCode
Algorithm Visualization
XOR Property: a ^ a = 0,  a ^ 0 = a

nums  = [3, 0, 1]   (n=3, missing=2)
index:   0  1  2

XOR all indices 0..n: 0^1^2^3 = 0
XOR all values:       3^0^1   = 2

Step-by-step:
  result = 0
  i=0: result ^= 0 ^= 3  -> 3
  i=1: result ^= 1 ^= 0  -> 2
  i=2: result ^= 2 ^= 1  -> 1
  i=3: result ^= 3       -> 2  (answer)
Basic Solution
class Solution:
    def missingNumber(self, nums):
        n = len(nums)
        expected = n * (n + 1) // 2
        return expected - sum(nums)
Optimized Solution (XOR)
class Solution:
    def missingNumber(self, nums):
        result = len(nums)
        for i, val in enumerate(nums):
            result ^= i ^ val
        return result
Complexity Analysis
VersionTimeSpaceReason
Basic (Sum)O(n)O(1)Single pass sum; arithmetic in O(1) space
Optimized (XOR)O(n)O(1)Single loop XOR; avoids potential overflow on large n
#201 Bitwise AND of Numbers Range Medium by LeetCode
Algorithm Visualization
Goal: AND of all numbers in [left, right]

Key: only the common prefix bits survive AND.
Any differing bit is zeroed by consecutive numbers.

Example: left=5, right=7
  5 = 101
  6 = 110
  7 = 111
AND = 100  (only common prefix survives)

Shift approach:
  5(101), 7(111) -> shift -> 2(10),3(11) -> shift -> 1,1 done
  shift count = 2
  result = 1 << 2 = 4
Basic Solution
class Solution:
    def rangeBitwiseAnd(self, left, right):
        mask = ~0
        while (left & mask) != (right & mask):
            mask <<= 1
        return right & mask
Optimized Solution (Right Shift)
class Solution:
    def rangeBitwiseAnd(self, left, right):
        shift = 0
        while left != right:
            left >>= 1
            right >>= 1
            shift += 1
        return left << shift
Complexity Analysis
VersionTimeSpaceReason
Basic (Mask)O(log n)O(1)At most 32 iterations (integer bit width)
Optimized (Shift)O(log n)O(1)Shifts until common prefix found; cleaner logic
#260 Single Number III Hard by LeetCode
Algorithm Visualization
Two numbers appear once; rest twice. Find both.

nums = [1, 2, 1, 3, 2, 5]  ->  answer: [3, 5]

Step 1: XOR all -> result is a XOR b
  1^2^1^3^2^5 = 3^5 = 6  (binary: 110)

Step 2: Find lowest set bit (differs between 3 and 5)
  diff = 6 & (-6) = 010

Step 3: Split by diff bit, XOR each group
  group A (bit clear): 1,1,2,2,5 -> XOR = 5
  group B (bit set):   3         -> XOR = 3
Basic Solution
from collections import Counter
class Solution:
    def singleNumber(self, nums):
        count = Counter(nums)
        return [k for k, v in count.items() if v == 1]
Optimized Solution (XOR + Bit Grouping)
class Solution:
    def singleNumber(self, nums):
        xor = 0
        for n in nums:
            xor ^= n
        diff = xor & (-xor)
        a = 0
        for n in nums:
            if n & diff:
                a ^= n
        return [a, xor ^ a]
Complexity Analysis
VersionTimeSpaceReason
Basic (Counter)O(n)O(n)Hash map stores all element counts
Optimized (XOR)O(n)O(1)Two linear passes; only 3 integer variables used
