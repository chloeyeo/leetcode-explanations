Two Sum II - Input Array Is Sorted
🔍 High Level
정렬된 배열에서 두 수의 합이 target이 되는 두 인덱스를 찾는 문제. 양 끝 포인터로 O(n)에 해결.
numbers = [2, 7, 11, 15], target = 9 L R ↓ ↓ [2, 7, 11, 15] → 2+15=17 (크다) → R-- L R ↓ ↓ [2, 7, 11, 15] → 2+11=13 (크다) → R-- L R ↓ ↓ [2, 7, 11, 15] → 2+7=9 ✅ → [1,2]
💻 기본 풀이
🐍 Python
def twoSum(numbers, target):
    left = 0
    right = len(numbers) - 1
    while left < right:
        current_sum = numbers[left] + numbers[right]
        if current_sum == target:
            return [left + 1, right + 1]  # 1-indexed
        elif current_sum < target:
            left += 1
        else:
            right -= 1
⚡ 최적화 버전 OPTIMAL
🐍 Python
def twoSum(numbers, target):
    l, r = 0, len(numbers) - 1
    while l < r:
        s = numbers[l] + numbers[r]
        if s == target: return [l+1, r+1]
        if s < target: l += 1
        else: r -= 1
