#643 Maximum Average Subarray I Easy — by LeetCode
📊 알고리즘 시각화
nums = [1, 12, -5, -6, 50, 3],  k = 4
Window slides right one step at a time:

idx:  0   1   2   3   4   5
    [ 1  12  -5  -6  50   3]
     [___________]
     sum= 2   avg= 0.5

         [___________]
          sum=51  avg=12.75  ← MAX

              [___________]
               sum=42  avg=10.5

Answer: 12.75
💻 기본 풀이 (Brute Force)
Basic Solution — O(n·k) time
class Solution:
    def findMaxAverage(self, nums: list[int], k: int) -> float:
        max_avg = float('-inf')
        for i in range(len(nums) - k + 1):
            # compute sum of window [i..i+k-1]
            window_sum = sum(nums[i : i + k])
            max_avg = max(max_avg, window_sum / k)
        return max_avg
⚡ 최적화 풀이 (Sliding Window)
Optimized — O(n) time, O(1) space
class Solution:
    def findMaxAverage(self, nums: list[int], k: int) -> float:
        # build initial window of size k
        window_sum = sum(nums[:k])
        max_sum = window_sum
        for i in range(k, len(nums)):
            # slide: add next element, remove leftmost
            window_sum += nums[i] - nums[i - k]
            max_sum = max(max_sum, window_sum)
        return max_sum / k
📈 복잡도 분석
버전시간공간이유
BasicO(n·k)O(1)각 위치마다 sum()이 k번 반복
OptimizedO(n)O(1)각 스텝에서 +1 -1만 수행, 전체 배열 1회 순회
#3 Longest Substring Without Repeating Characters Medium — by LeetCode
📊 알고리즘 시각화
s = "abcabcbb"
Dynamic window expands right, shrinks left on duplicate:

left                right
  a  b  c  a  b  c  b  b
  [a]              len=1
  [a  b]           len=2
  [a  b  c]        len=3
     [b  c  a]     'a' seen — shrink from left
        [c  a  b]  len=3
           [a  b  c] len=3  MAX=3
💻 기본 풀이
Basic Solution — O(n^2) time
class Solution:
    def lengthOfLongestSubstring(self, s: str) -> int:
        max_len = 0
        n = len(s)
        for i in range(n):
            seen = set()
            for j in range(i, n):
                if s[j] in seen:
                    break
                seen.add(s[j])
                max_len = max(max_len, j - i + 1)
        return max_len
⚡ 최적화 풀이 (Sliding Window + HashMap)
Optimized — O(n) time, O(min(n,m)) space
class Solution:
    def lengthOfLongestSubstring(self, s: str) -> int:
        char_index = {}   # char -> last seen index
        left = 0
        max_len = 0
        for right, ch in enumerate(s):
            if ch in char_index and char_index[ch] >= left:
                # jump left past the duplicate
                left = char_index[ch] + 1
            char_index[ch] = right
            max_len = max(max_len, right - left + 1)
        return max_len
📈 복잡도 분석
버전시간공간이유
BasicO(n^2)O(min(n,m))이중 for 루프로 모든 시작점 탐색
OptimizedO(n)O(min(n,m))right 포인터 한 번만 순회, left는 딕셔너리로 O(1) 점프
#76 Minimum Window Substring Hard — by LeetCode
📊 알고리즘 시각화
s = "ADOBECODEBANC",  t = "ABC"
need = {A:1, B:1, C:1}

Expand right until window has all of t,
then contract left to minimize:

 A  D  O  B  E  C  O  D  E  B  A  N  C
[A  D  O  B  E  C] have ABC -> len=6
   [D  O  B  E  C] missing A
      [O..B  A] have ABC -> len=9 then shrink
         [B  A  N  C] have ABC -> len=4
            [A  N  C] missing B...

Best window: "BANC" (len=4)
💻 기본 풀이
Basic Solution — O(n^2) time
from collections import Counter

class Solution:
    def minWindow(self, s: str, t: str) -> str:
        need = Counter(t)
        result = ""
        for i in range(len(s)):
            window = Counter()
            for j in range(i, len(s)):
                window[s[j]] += 1
                if all(window[c] >= need[c] for c in need):
                    candidate = s[i:j+1]
                    if not result or len(candidate) < len(result):
                        result = candidate
                    break
        return result
⚡ 최적화 풀이 (Sliding Window + formed count)
Optimized — O(|s|+|t|) time, O(|s|+|t|) space
from collections import Counter

class Solution:
    def minWindow(self, s: str, t: str) -> str:
        if not t or not s: return ""
        need = Counter(t)
        required = len(need)      # unique chars needed
        window = {}
        formed = 0              # unique chars satisfied
        left = 0
        ans = float('inf'), 0, 0  # (len, l, r)
        for right, ch in enumerate(s):
            window[ch] = window.get(ch, 0) + 1
            if ch in need and window[ch] == need[ch]:
                formed += 1
            # contract while window is valid
            while left <= right and formed == required:
                if right - left + 1 < ans[0]:
                    ans = right - left + 1, left, right
                lc = s[left]
                window[lc] -= 1
                if lc in need and window[lc] < need[lc]:
                    formed -= 1
                left += 1
        return "" if ans[0] == float('inf') else s[ans[1]:ans[2]+1]
📈 복잡도 분석
버전시간공간이유
BasicO(n^2*|t|)O(|s|+|t|)이중 루프 + 매번 all() 체크로 O(|t|)
OptimizedO(|s|+|t|)O(|s|+|t|)formed 카운터로 O(1) 유효성 체크, left/right 각 최대 n번 이동
