LeetCode #643 — Maximum Average Subarray I
by LeetCode
Easy
🔭 Algorithm Visualization
nums = [1, 12, -5, -6, 50, 3],  k = 4

Window slides one step at a time:
[1, 12, -5, -6] 50   3   → sum=2,  avg=0.5
 1 [12, -5, -6, 50]  3   → sum=51, avg=12.75  ← MAX
 1  12 [-5, -6, 50,  3]  → sum=42, avg=10.5

Window: add right, remove left
 ┌─────────────────────┐
 │  [l ←───── r]       │
 │   remove  add new   │
 └─────────────────────┘
💻 기본 풀이 (Python)
def findMaxAverage(nums, k):
    # 첫 윈도우 합 계산
    window_sum = sum(nums[:k])
    max_sum = window_sum

    for i in range(k, len(nums)):
        # 오른쪽 추가, 왼쪽 제거
        window_sum += nums[i] - nums[i - k]
        max_sum = max(max_sum, window_sum)

    return max_sum / k
⚡ 최적화 버전 (Python)
def findMaxAverage(nums, k):
    # 나눗셈 없이 정수 비교로 최적화
    s = sum(nums[:k])
    best = s
    for i in range(k, len(nums)):
        s += nums[i] - nums[i-k]
        if s > best:
            best = s
    return best / k
📊 복잡도 분석
구분복잡도이유
TimeO(n)배열을 한 번 순회. 초기 합 O(k) + 슬라이딩 O(n-k) = O(n)
SpaceO(1)추가 자료구조 없이 변수 2개(s, best)만 사용
LeetCode #3 — Longest Substring Without Repeating Characters
by LeetCode
Medium
🔭 Algorithm Visualization
s = "abcabcbb"

Variable-size sliding window:
 a b c a b c b b
 ↑               l=0
 ↑   ↑           "abc" len=3  ← MAX
     ↑   ↑       "cab" len=3
         ↑ ↑     "cb"  len=2 (dup 'b')

Char Set:  {a,b,c}
When dup found: shrink from left until removed
 ┌──────────────────────────────────┐
 │  l ──────────────────► r          │
 │  shrink ←          expand →      │
└──────────────────────────────────┘
💻 기본 풀이 (Python)
def lengthOfLongestSubstring(s):
    char_set = set()
    left = 0
    max_len = 0

    for right in range(len(s)):
        # 중복 문자가 있으면 왼쪽 포인터 이동
        while s[right] in char_set:
            char_set.remove(s[left])
            left += 1
        char_set.add(s[right])
        max_len = max(max_len, right - left + 1)

    return max_len
⚡ 최적화 버전 (Python)
def lengthOfLongestSubstring(s):
    # 딕셔너리로 인덱스 저장 → O(1)로 left 점프
    char_idx = {}
    left = 0
    best = 0

    for right, ch in enumerate(s):
        if ch in char_idx and char_idx[ch] >= left:
            # 중복 위치 바로 다음으로 점프
            left = char_idx[ch] + 1
        char_idx[ch] = right
        best = max(best, right - left + 1)

    return best

SpaceO(min(n,m))O(min(n,m))set/dict 크기 = 고유 문자 수 m (ASCII 128)
LeetCode #76 — Minimum Window Substring
by LeetCode
Hard
🔭 Algorithm Visualization
s = "ADOBECODEBANC",  t = "ABC"

need = {A:1, B:1, C:1},  have = {}

Expand right until valid, then shrink left:

 A D O B E C O D E B A  N  C
 ↑                         → expand
 A D O B E C               formed! {A,B,C} ✓
   ↑                       → shrink (remove A)
   D O B E C               missing A ✗
   D O B E C O D E B  A    → expand
                     ↑     shrink
               B  A  N  C  → "BANC" len=4 ← MIN
💻 기본 풀이 (Python)
from collections import Counter

def minWindow(s, t):
    if not t or not s:
        return ""

    need = Counter(t)
    have, required = {}, len(need)
    formed = 0
    left = 0
    best = float("inf"), 0, 0

    for right, ch in enumerate(s):
        have[ch] = have.get(ch, 0) + 1
        if ch in need and have[ch] == need[ch]:
            formed += 1

        # 유효 윈도우이면 최소화 시도
        while formed == required:
            if (right - left + 1) < best[0]:
                best = (right - left + 1, left, right)
            lc = s[left]
            have[lc] -= 1
            if lc in need and have[lc] < need[lc]:
                formed -= 1
            left += 1

    return "" if best[0] == float("inf") else s[best[1]:best[2]+1]
⚡ 최적화 버전 (Python)
from collections import Counter

def minWindow(s, t):
    # 필터링: t에 있는 문자만 처리 (s가 길 때 효과적)
    filtered = [(i, c) for i, c in enumerate(s) if c in Counter(t)]
    need = Counter(t)
    have = {}
    formed = 0
    required = len(need)
    l = 0
    best = float("inf"), 0, 0

    for r in range(len(filtered)):
        ch = filtered[r][1]
        have[ch] = have.get(ch, 0) + 1
        if have[ch] == need[ch]:
            formed += 1
        while formed == required:
            lo, hi = filtered[l][0], filtered[r][0]
            if hi - lo + 1 < best[0]:
                best = hi - lo + 1, lo, hi
            lc = filtered[l][1]
            have[lc] -= 1
            if have[lc] < need[lc]:
                formed -= 1
            l += 1

    return "" if best[0] == float("inf") else s[best[1]:best[2]+1]

구분기본최적홐이유

SpaceO(|t|)O(|s|+|t|)최적홐: filtered 배열 추가 O(|s|), 실제 긴 문자열엔 시간 이득
