#997 Find the Town Judge
Easy
Algorithm Visualization
Town: N=4, trust=[[1,3],[1,4],[2,3],[2,4],[4,3]] 1 --trusts--> 3 indegree[3]=3 OK 2 --trusts--> 3 outdegree[3]=0 OK Judge = node where indegree==N-1 AND outdegree==0 --> Node 3 is the JUDGE
Basic Solution
class Solution: def findJudge(self, n, trust): given = [0]*(n+1); recd = [0]*(n+1) for a,b in trust: given[a]+=1; recd[b]+=1 for i in range(1,n+1): if given[i]==0 and recd[i]==n-1: return i return -1
Optimized Solution
class Solution: def findJudge(self, n, trust): score = [0]*(n+1) for a,b in trust: score[a]-=1; score[b]+=1 for i in range(1,n+1): if score[i]==n-1: return i return -1
Complexity Analysis
MetricBasicOptimizedReason
TimeO(E+N)O(E+N)One pass edges + one pass nodes
SpaceO(N)O(N)Basic: 2 arrays; Optimized: 1 score array
KeyIn/Out degreeJudge: indegree=N-1, outdegree=0 → net score=N-1
#207 Course Schedule
Medium
Algorithm Visualization
numCourses=4, prerequisites=[[1,0],[2,0],[3,1],[3,2]] 0 --> 1 --> 3 0 --> 2 --> 3 Kahn's BFS: in-degree=[0,1,1,2] Queue:[0] -> [1,2] -> [3] -> done Processed=4==numCourses -> NO CYCLE
Basic Solution (DFS)
class Solution: def canFinish(self, n, prereqs): g=[[] for _ in range(n)] for a,b in prereqs: g[b].append(a) st=[0]*n def dfs(v): if st[v]==1: return False if st[v]==2: return True st[v]=1 for u in g[v]: if not dfs(u): return False st[v]=2; return True return all(dfs(i) for i in range(n))
Optimized Solution (Kahn's BFS)
from collections import deque class Solution: def canFinish(self, n, prereqs): g=[[] for _ in range(n)]; ind=[0]*n for a,b in prereqs: g[b].append(a); ind[a]+=1 q=deque(i for i in range(n) if ind[i]==0); cnt=0 while q: v=q.popleft(); cnt+=1 for u in g[v]: ind[u]-=1 if ind[u]==0: q.append(u) return cnt==n

SpaceO(V+E)O(V+E)DFS: call stack; BFS: queue+graph

#127 Word Ladder
Hard
Algorithm Visualization
beginWord="hit" endWord="cog" wordList=["hot","dot","dog","lot","log","cog"] Level 1: hit Level 2: hot (i->o) Level 3: dot, lot Level 4: dog, log Level 5: cog <-- FOUND! Answer=5 Bi-BFS: expand smaller frontier each step Forward: hit->hot->dot/lot Backward: cog->dog/log->dot/lot (meet!)
Basic Solution (BFS)
from collections import deque class Solution: def ladderLength(self, bw, ew, wl): ws=set(wl) if ew not in ws: return 0 q=deque([(bw,1)]); seen={bw} while q: w,d=q.popleft() for i in range(len(w)): for c in 'abcdefghijklmnopqrstuvwxyz': nw=w[:i]+c+w[i+1:] if nw==ew: return d+1 if nw in ws and nw not in seen: seen.add(nw); q.append((nw,d+1)) return 0
Optimized Solution (Bidirectional BFS)
class Solution: def ladderLength(self, bw, ew, wl): ws=set(wl) if ew not in ws: return 0 f,b={bw},{ew}; seen={bw,ew}; d=1 while f: if len(f)>len(b): f,b=b,f nf=set() for w in f: for i in range(len(w)): for c in 'abcdefghijklmnopqrstuvwxyz': nw=w[:i]+c+w[i+1:] if nw in b: return d+1 if nw in ws and nw not in seen: seen.add(nw); nf.add(nw) f=nf; d+=1 return 0
Complexity Analysis
MetricBFSBi-BFSReason
TimeO(M²×N)O(M²×N/2)M=word len, N=list size; bi-BFS halves depth
SpaceO(M×N)O(M×N)Queue and visited set of transformed words
KeyBFS layersMeet middleSmaller frontier → far fewer nodes explored
