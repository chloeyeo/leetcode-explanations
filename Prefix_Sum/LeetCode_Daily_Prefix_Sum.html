LeetCode #303 — Range Sum Query - Immutable Easy by LeetCode

📊 Algorithm Visualization

nums = [-2, 0, 3, -5, 2, -1] Build prefix sum array: index: 0 1 2 3 4 5 nums: [-2, 0, 3, -5, 2, -1] prefix:[ 0, -2, -2, 1, -4, -2, -3] prefix[0] = 0 (sentinel) prefix[i] = prefix[i-1] + nums[i-1] Query: sumRange(0, 2) = prefix[3] - prefix[0] = 1 - 0 = 1 Query: sumRange(2, 5) = prefix[6] - prefix[2] = -3 - (-2) = -1 Query: sumRange(0, 5) = prefix[6] - prefix[0] = -3 - 0 = -3 Time O(1) per query after O(n) build!

🐍 Basic Solution

class NumArray:

    def __init__(self, nums):
        self.prefix = [0] * (len(nums) + 1)
        for i in range(len(nums)):
            self.prefix[i + 1] = self.prefix[i] + nums[i]

    def sumRange(self, left, right):
        return self.prefix[right + 1] - self.prefix[left]

⚡ Optimized Solution

import itertools

class NumArray:

    def __init__(self, nums):
        self.prefix = [*itertools.accumulate(nums, initial=0)]

    def sumRange(self, left, right):
        return self.prefix[right + 1] - self.prefix[left]

LeetCode #560 — Subarray Sum Equals K Medium by LeetCode

📊 Algorithm Visualization

nums = [1, 1, 1], k = 2 Track running prefix sum + hashmap of seen sums: Step 0: prefix=0 map={0:1} count=0 Step 1: prefix=1 map={0:1,1:1} need=1-2=-1 not in map, count=0 Step 2: prefix=2 map={0:1,1:1,2:1} need=2-2=0 map[0]=1, count=1 Step 3: prefix=3 need=3-2=1 map[1]=1, count=2 Answer: 2 Key insight: subarray(i,j) sum = prefix[j] - prefix[i-1] If prefix[j] - k exists in map, found a valid subarray!

🐍 Basic Solution

def subarraySum(nums, k):
    count = 0
    for i in range(len(nums)):
        total = 0
        for j in range(i, len(nums)):
            total += nums[j]
            if total == k:
                count += 1
    return count

⚡ Optimized Solution

from collections import defaultdict

def subarraySum(nums, k):
    count = 0
    prefix = 0
    seen = defaultdict(int)
    seen[0] = 1

    for num in nums:
        prefix += num
        count += seen[prefix - k]
        seen[prefix] += 1

    return count

LeetCode #862 — Shortest Subarray with Sum at Least K Hard by LeetCode

📊 Algorithm Visualization

nums = [2, -1, 2], k = 3 prefix = [0, 2, 1, 3] (negative numbers allowed!) Monotonic deque approach: - Deque stores indices of prefix sums in increasing order - For each j, pop left while prefix[j]-prefix[deque[0]] >= k j=0: prefix=0 deque=[0] j=1: prefix=2 deque=[0,1] prefix[1]-prefix[0]=2 < 3 j=2: prefix=1 1<2, pop 1 deque=[0,2] j=3: prefix=3 prefix[3]-prefix[0]=3>=3, ans=min(inf,3-0)=3, pop 0 deque=[2] Answer: 3 Why deque? Negative numbers break sliding window! Deque maintains monotone prefix for O(n) solution.

🐍 Basic Solution

def shortestSubarray(nums, k):
    n = len(nums)
    prefix = [0] * (n + 1)
    for i in range(n):
        prefix[i + 1] = prefix[i] + nums[i]

    ans = float('inf')
    for i in range(n + 1):
        for j in range(i + 1, n + 1):
            if prefix[j] - prefix[i] >= k:
                ans = min(ans, j - i)
    return ans if ans != float('inf') else -1

⚡ Optimized Solution

from collections import deque

def shortestSubarray(nums, k):
    n = len(nums)
    prefix = [0] * (n + 1)
    for i in range(n):
        prefix[i + 1] = prefix[i] + nums[i]

    ans = float('inf')
    dq = deque()

    for j in range(n + 1):
        while dq and prefix[j] - prefix[dq[0]] >= k:
            ans = min(ans, j - dq.popleft())
        while dq and prefix[j] <= prefix[dq[-1]]:
            dq.pop()
        dq.append(j)

    return ans if ans != float('inf') else -1
