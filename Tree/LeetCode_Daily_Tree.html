LeetCode #104 вҖ” Maximum Depth of Binary Tree Easy by LeetCode
рҹҺЁ м•Ңкі лҰ¬мҰҳ мӢңк°Ғнҷ”
Binary Tree: DFS (мһ¬к·Җ) нғҗмғү: 3 visit(3) / \ / \ 9 20 visit(9) visit(20) / \ | / \ 15 7 depth=1 visit(15) visit(7) | | depth=2 depth=2 \ max(1, max(2,2)) + 1 = 3 кІ°кіј: maxDepth = 3
рҹ“қ кё°ліё н’Җмқҙ (мһ¬к·Җ DFS)
class Solution: def maxDepth(self, root): # лІ мқҙмҠӨ мјҖмқҙмҠӨ: л…ёл“ңк°Җ м—Ҷмңјл©ҙ к№Ҡмқҙ 0 if not root: return 0 # мҷјмӘҪ/мҳӨлҘёмӘҪ м„ңлёҢнҠёлҰ¬мқҳ мөңлҢҖ к№Ҡмқҙ мһ¬к·Җ кі„мӮ° left_depth = self.maxDepth(root.left) right_depth = self.maxDepth(root.right) # лҚ” к№ҠмқҖ мӘҪ + нҳ„мһ¬ л…ёл“ң 1 return max(left_depth, right_depth) + 1
вҡЎ мөңм Ғнҷ” лІ„м „ (л°ҳліө BFS)
from collections import deque class Solution: def maxDepth(self, root): if not root: return 0 queue, depth = deque([root]), 0 while queue: # нҳ„мһ¬ л ҲлІЁмқҳ лӘЁл“  л…ёл“ң мІҳлҰ¬ for _ in range(len(queue)): node = queue.popleft() if node.left: queue.append(node.left) if node.right: queue.append(node.right) depth += 1 return depth

LeetCode #102 вҖ” Binary Tree Level Order Traversal Medium by LeetCode
рҹҺЁ м•Ңкі лҰ¬мҰҳ мӢңк°Ғнҷ”
3 BFS Queue мғҒнғң ліҖнҷ”: / \ 9 20 Level 0: [3] вҶ’ [[3]] / \ Level 1: [9, 20] вҶ’ [[3],[9,20]] 15 7 Level 2: [15, 7] вҶ’ [[3],[9,20],[15,7]] кІ°кіј: [[3], [9, 20], [15, 7]]
рҹ“қ кё°ліё н’Җмқҙ (BFS)
from collections import deque class Solution: def levelOrder(self, root): if not root: return [] result, queue = [], deque([root]) while queue: level_size = len(queue) level = [] for _ in range(level_size): node = queue.popleft() level.append(node.val) if node.left: queue.append(node.left) if node.right: queue.append(node.right) result.append(level) return result

class Solution: def levelOrder(self, root): result = [] def dfs(node, level): if not node: return # нҳ„мһ¬ л ҲлІЁ лҰ¬мҠӨнҠёк°Җ м—Ҷмңјл©ҙ мғҲлЎң м¶”к°Җ if len(result) == level: result.append([]) result[level].append(node.val) dfs(node.left, level + 1) dfs(node.right, level + 1) dfs(root, 0) return result

LeetCode #124 вҖ” Binary Tree Maximum Path Sum Hard by LeetCode
рҹҺЁ м•Ңкі лҰ¬мҰҳ мӢңк°Ғнҷ”
-10 кІҪлЎң нӣ„ліt: / \ 9 вҶ’ 9 9 20 15вҶ’20вҶ’7 вҶ’ 42 вң… MAX / \ 9вҶ’-10вҶ’20 вҶ’ 19 15 7 gain(-10) = max(gain(9), 0) + max(gain(20), 0) + (-10) = max(9,0) + max(15+20, 0) + (-10) = 9+35-10 = 34 global_max к°ұмӢ : max(42, 34) = 42
рҹ“қ кё°ліё н’Җмқҙ (нӣ„мң„ мҲңнҡҢ)
class Solution: def maxPathSum(self, root): self.max_sum = float('-inf') def gain(node): if not node: return 0 # мқҢмҲҳ кё°м—¬лҠ” 0мңјлЎң мІҳлҰ¬ (н•ҙлӢ№ кІҪлЎң нҸ¬кё°) left_gain = max(gain(node.left), 0) right_gain = max(gain(node.right), 0) # мқҙ л…ёл“ңлҘј кјӯм§“м җмңјлЎң н•ҳлҠ” кІҪлЎң н•© path_sum = node.val + left_gain + right_gain self.max_sum = max(self.max_sum, path_sum) # л¶ҖлӘЁлЎң мҳ¬лқјк°Ҳ л•ҢлҠ” н•ң л°©н–Ҙл§Ң м„ нғқ return node.val + max(left_gain, right_gain) gain(root) return self.max_sum
вҡЎ мөңм Ғнҷ” лІ„м „ (nonlocal нҷңмҡ©)
class Solution: def maxPathSum(self, root): max_sum = [float('-inf')] # лҰ¬мҠӨнҠёлЎң nonlocal лҢҖмІҙ def dfs(node): if not node: return 0 l = max(dfs(node.left), 0) r = max(dfs(node.right), 0) max_sum[0] = max(max_sum[0], node.val + l + r) return node.val + max(l, r) dfs(root) return max_sum[0]
