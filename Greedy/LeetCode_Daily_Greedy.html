Algorithm Visualization
Greed Factor: [1, 2, 3] Cookie Size: [1, 1] Sort both arrays: g: [1, 2, 3] s: [1, 1] i=0(g=1), j=0(s=1): 1>=1 OK assign, count=1, i++, j++ i=1(g=2), j=1(s=1): 1<2 NG skip j exhausted -> result = 1 Rule: smallest valid cookie to smallest needy child
Basic Solution
class Solution: def findContentChildren(self, g, s): g.sort(); s.sort() child = cookie = 0 while child < len(g) and cookie < len(s): if s[cookie] >= g[child]: child += 1 cookie += 1 return child
Optimized Solution
class Solution: def findContentChildren(self, g, s): g.sort(); s.sort() i = j = 0 while i < len(g) and j < len(s): if s[j] >= g[i]: i += 1 j += 1 return i

LeetCode #435 - Non-overlapping Intervals Medium

by LeetCode

Algorithm Visualization
intervals = [[1,2],[2,3],[3,4],[1,3]] Sort by END time: [1,2] [2,3] [1,3] [3,4] Track kept end: [1,2] keep -> end=2 removed=0 [2,3] 2>=2 keep -> end=3 removed=0 [1,3] 1<3 skip (overlap) removed=1 [3,4] 3>=3 keep -> end=4 Answer: 1 removal
Basic Solution
class Solution: def eraseOverlapIntervals(self, intervals): if not intervals: return 0 intervals.sort(key=lambda x: x[1]) removed = 0; end = intervals[0][1] for i in range(1, len(intervals)): if intervals[i][0] < end: removed += 1 else: end = intervals[i][1] return removed
Optimized Solution
class Solution: def eraseOverlapIntervals(self, intervals): # Activity Selection: count max non-overlapping then subtract intervals.sort(key=lambda x: x[1]) keep = 1; end = intervals[0][1] for s, e in intervals[1:]: if s >= end: keep += 1; end = e return len(intervals) - keep
Complexity Analysis
SolutionTimeSpaceReason
BasicO(n log n)O(1)Sort + single linear scan
OptimizedO(n log n)O(1)Same; count kept then subtract
LeetCode #45 - Jump Game II Hard

by LeetCode

Algorithm Visualization
nums = [2, 3, 1, 1, 4] idx: 0 1 2 3 4 BFS-level greedy (each level = 1 jump): Level 0: idx0 -> far=2, reach level end: jumps=1, end=2 Level 1: idx1: 1+3=4(new far!), idx2: 2+1=3 reach level end(idx2): jumps=2, end=4 end(4) >= last(4) DONE -> Answer: 2 jumps
Basic Solution
class Solution: def jump(self, nums): jumps = curr_end = farthest = 0 for i in range(len(nums) - 1): farthest = max(farthest, i + nums[i]) if i == curr_end: jumps += 1; curr_end = farthest if curr_end >= len(nums) - 1: break return jumps
Optimized Solution
class Solution: def jump(self, nums): jumps = cur = far = 0 for i in range(len(nums) - 1): far = max(far, i + nums[i]) if i == cur: jumps += 1; cur = far return jumps
Complexity Analysis
SolutionTimeSpaceReason
BasicO(n)O(1)Single pass; early break when end reachable
OptimizedO(n)O(1)Same; early-exit removed for brevity
