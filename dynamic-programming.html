#70
Climbing Stairs
🔍 High Level
한 번에 1칸 또는 2칸씩 오를 때, n번째 계단에 도달하는 방법의 수. 점화식 dp[i] = dp[i-1] + dp[i-2] (피보나치).
n = 5 dp[1] = 1 (1) dp[2] = 2 (1+1, 2) dp[3] = 3 = dp[2] + dp[1] dp[4] = 5 = dp[3] + dp[2] dp[5] = 8 = dp[4] + dp[3] ✅ 상태: dp[i] = i번째 계단에 도달하는 경우의 수 전이: dp[i] = dp[i-1] + dp[i-2]
💻 기본 풀이
🐍 Python
def climbStairs(n):
    if n <= 2: return n
    dp = [0] * (n + 1)
    dp[1], dp[2] = 1, 2
    for i in range(3, n + 1):
        dp[i] = dp[i - 1] + dp[i - 2]
    return dp[n]
⚡ 최적화 버전 OPTIMAL
배열 대신 변수 2개만 사용 → 공간 O(1)로 축소.
🐍 Python
def climbStairs(n):
    a, b = 1, 1
    for _ in range(n - 1):
        a, b = b, a + b
    return b

⏱ 시간

💾 공간

#322
Coin Change
🔍 High Level
동전 종류 coins로 금액 amount를 만들기 위한 최소 동전 개수. 만들 수 없으면 -1. 1차원 DP로 모든 금액에 대해 최솟값 탐색.
coins = [1, 2, 5], amount = 11 dp[0] = 0 dp[1] = 1 (1) dp[2] = 1 (2) dp[3] = 2 (1+2) dp[4] = 2 (2+2) dp[5] = 1 (5) ... dp[11] = 3 (5+5+1) ✅ 전이: dp[i] = min(dp[i - c] + 1) for c in coins
💻 기본 풀이
🐍 Python
def coinChange(coins, amount):
    INF = float('inf')
    dp = [INF] * (amount + 1)
    dp[0] = 0
    for i in range(1, amount + 1):
        for c in coins:
            if i - c >= 0:
                dp[i] = min(dp[i], dp[i - c] + 1)
    return dp[amount] if dp[amount] != INF else -1
⚡ 최적화 버전 OPTIMAL
동전을 한 번 정렬해두면 i < c인 동전은 즉시 break 가능 → 평균적으로 빠름.
🐍 Python
def coinChange(coins, amount):
    coins.sort()
    dp = [0] + [amount + 1] * amount
    for i in range(1, amount + 1):
        for c in coins:
            if c > i: break
            dp[i] = min(dp[i], dp[i - c] + 1)
    return dp[amount] if dp[amount] <= amount else -1
📊 복잡도 분석
coins.sort()
동전 정렬 → O(k log k), k = 동전 수
dp = [0] + [amount+1]*amount
dp 테이블 초기화 → O(amount)
for i in range(1, amount+1)
바깥 루프 → amount번
for c in coins
안쪽 루프 → 동전 수만큼 (최악 k)
⏱ 시간
O(n·k)
n = amount, k = 동전 개수
💾 공간
O(n)
dp 테이블 길이 = amount+1

#72
Edit Distance
🔍 High Level
word1을 word2로 만드는 최소 편집 횟수 (삽입/삭제/교체). 2차원 DP — dp[i][j] = word1의 앞 i자 → word2의 앞 j자로 만드는 최소 비용.
word1 = "horse", word2 = "ros" "" r o s "" [0, 1, 2, 3] h [1, 1, 2, 3] o [2, 2, 1, 2] r [3, 2, 2, 2] s [4, 3, 3, 2] e [5, 4, 4, 3] ✅ 전이: 같으면: dp[i][j] = dp[i-1][j-1] 다르면: dp[i][j] = 1 + min( dp[i-1][j], # 삭제 dp[i][j-1], # 삽입 dp[i-1][j-1]) # 교체
💻 기본 풀이
🐍 Python
def minDistance(word1, word2):
    m, n = len(word1), len(word2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    for i in range(m + 1): dp[i][0] = i
    for j in range(n + 1): dp[0][j] = j
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if word1[i-1] == word2[j-1]:
                dp[i][j] = dp[i-1][j-1]
            else:
                dp[i][j] = 1 + min(dp[i-1][j], dp[i][j-1], dp[i-1][j-1])
    return dp[m][n]
⚡ 최적화 버전 OPTIMAL
현재 행 계산에 직전 행만 필요 → 1차원 배열 2개로 공간 O(min(m,n))로 축소.
🐍 Python
def minDistance(word1, word2):
    if len(word1) < len(word2):
        word1, word2 = word2, word1
    m, n = len(word1), len(word2)
    prev = list(range(n + 1))
    for i in range(1, m + 1):
        curr = [i] + [0] * n
        for j in range(1, n + 1):
            if word1[i-1] == word2[j-1]:
                curr[j] = prev[j-1]
            else:
                curr[j] = 1 + min(prev[j], curr[j-1], prev[j-1])
        prev = curr
    return prev[n]

바깥 루프 → m번
for j in range(1, n+1)
안쪽 루프 → n번
curr[j] = 1 + min(...)
세 후보 중 최솟값 → O(1)

행 교체 → 참조 변경 O(1)
⏱ 시간
O(m·n)
모든 (i,j) 쌍 1회씩 채움
💾 공간
