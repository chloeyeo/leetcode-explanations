LeetCode #766 — Toeplitz Matrix

by LeetCode
Easy

🎨 Algorithm Visualization

Matrix: [[1,2,3,4], [5,1,2,3], [9,5,1,2]] Diagonals (top-left to bottom-right): [1,1,1] ✓ [2,2,2] ✓ [3,3] ✓ [4] ✓ [5,5] ✓ [9] ✓ Check: matrix[i][j] == matrix[i-1][j-1]? (0,0)=1 (1,1)=1 ✓ (2,2)=1 ✓ (0,1)=2 (1,2)=2 ✓ (2,3)=2 ✓ → All diagonals match → True

🐍 Basic Solution (Python)

class Solution:
    def isToeplitzMatrix(self, matrix):
        rows = len(matrix)
        cols = len(matrix[0])
        for i in range(1, rows):
            for j in range(1, cols):
                if matrix[i][j] != matrix[i-1][j-1]:
                    return False
        return True

⚡ Optimized Solution (Python)

class Solution:
    def isToeplitzMatrix(self, matrix):
        # Compare each row (except first) with previous row shifted right
        return all(
            r1[:-1] == r2[1:]
            for r1, r2 in zip(matrix, matrix[1:])
        )

LeetCode #54 — Spiral Matrix

by LeetCode
Medium

🎨 Algorithm Visualization

Matrix: [[1,2,3], [4,5,6], [7,8,9]] Peel layers outward: → Right: 1 2 3 → Down: 6 9 → Left: 8 7 → Up: 4 → Center: 5 Result: [1,2,3,6,9,8,7,4,5] Boundaries shrink each pass: top↓ bottom↑ left→ right←

🐍 Basic Solution (Python)

class Solution:
    def spiralOrder(self, matrix):
        result = []
        top, bottom = 0, len(matrix) - 1
        left, right = 0, len(matrix[0]) - 1
        while top <= bottom and left <= right:
            # Move right across top row
            for i in range(left, right + 1):
                result.append(matrix[top][i])
            top += 1
            # Move down right column
            for i in range(top, bottom + 1):
                result.append(matrix[i][right])
            right -= 1
            # Move left across bottom row
            if top <= bottom:
                for i in range(right, left - 1, -1):
                    result.append(matrix[bottom][i])
                bottom -= 1
            # Move up left column
            if left <= right:
                for i in range(bottom, top - 1, -1):
                    result.append(matrix[i][left])
                left += 1
        return result

⚡ Optimized Solution (Python)

class Solution:
    def spiralOrder(self, matrix):
        result = []
        while matrix:
            # Take top row
            result += matrix.pop(0)
            # Take right column of remaining rows
            if matrix and matrix[0]:
                for row in matrix:
                    result.append(row.pop())
            # Take bottom row reversed
            if matrix:
                result += matrix.pop()[::-1]
            # Take left column reversed
            if matrix and matrix[0]:
                for row in matrix[::-1]:
                    result.append(row.pop(0))
        return result

LeetCode #329 — Longest Increasing Path in a Matrix

by LeetCode
Hard

🎨 Algorithm Visualization

Matrix: [[9,9,4], [6,6,8], [2,1,1]] DFS + Memoization: Start at each cell, explore 4 dirs if value increases From 1>2>6>9 = length 4 ← LONGEST From 1>6>9 = length 3 From 1>8>9 = length 3 memo[i][j] = longest path starting from (i,j) memo table: [1, 1, 2] [2, 2, 3] [3, 4, 3] Answer: max(memo) = 4

🐍 Basic Solution (Python)

class Solution:
    def longestIncreasingPath(self, matrix):
        if not matrix: return 0
        rows, cols = len(matrix), len(matrix[0])
        memo = {}

        def dfs(i, j):
            if (i, j) in memo: return memo[(i, j)]
            best = 1
            for di, dj in [(0,1),(0,-1),(1,0),(-1,0)]:
                ni, nj = i + di, j + dj
                if 0 <= ni < rows and 0 <= nj < cols:
                    if matrix[ni][nj] > matrix[i][j]:
                        best = max(best, 1 + dfs(ni, nj))
            memo[(i, j)] = best
            return best

        return max(dfs(i, j) for i in range(rows) for j in range(cols))
