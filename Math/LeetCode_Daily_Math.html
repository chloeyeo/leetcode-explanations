LeetCode #9 — Palindrome Number

by LeetCode
Algorithm Visualization
Input: x = 121 Reverse second half: x = 121 | +-- last digit: 121 % 10 = 1 -> reversed = 1, x = 12 +-- last digit: 12 % 10 = 2 -> reversed = 12, x = 1 | +-- stop: x (1) <= reversed (12) | +-- x == reversed (1 == 1) PASS OR x == reversed // 10 (even length) Input: x = 123 -> reversed = 321, 12 != 1 FAIL Negative numbers: always False | Ends with 0 (except 0): always False
Basic Solution (Python)
class Solution:
    def isPalindrome(self, x: int) -> bool:
        # Convert to string and compare with its reverse
        if x < 0:
            return False
        s = str(x)
        return s == s[::-1]
Optimized Solution (No String Conversion)
class Solution:
    def isPalindrome(self, x: int) -> bool:
        if x < 0 or (x % 10 == 0 and x != 0):
            return False
        rev = 0
        while x > rev:
            rev = rev * 10 + x % 10
            x //= 10
        return x == rev or x == rev // 10

LeetCode #50 — Pow(x, n)

by LeetCode
Algorithm Visualization
Fast Exponentiation -- x^10 n=10 (even) -> x^10 = (x^5)^2 n=5 (odd) -> x^5 = x * (x^2)^2 n=2 (even)-> x^2 = (x^1)^2 n=1(odd)-> x^1 = x * (x^0)^2 n=0 -> return 1 Call tree depth = log2(n) levels Negative n: pow(x,n) = 1 / pow(x,-n)
Basic Solution (Python)
class Solution:
    def myPow(self, x: float, n: int) -> float:
        if n == 0: return 1.0
        if n < 0: x, n = 1/x, -n
        res = 1.0
        for _ in range(n): res *= x
        return res  # O(n) time
Optimized Solution (Fast Power)
class Solution:
    def myPow(self, x: float, n: int) -> float:
        def fp(b, e):
            if e == 0: return 1.0
            if e % 2 == 0:
                h = fp(b, e//2); return h*h
            return b * fp(b, e-1)
        if n < 0: x, n = 1/x, -n
        return fp(x, n)

LeetCode #149 — Max Points on a Line

by LeetCode
Algorithm Visualization
Points: [[1,1],[2,2],[3,3],[4,5]] Fix anchor [1,1], compute slopes to others: -> [2,2]: slope = 1/1 (gcd-normalized) -> [3,3]: slope = 1/1 <- SAME LINE! -> [4,5]: slope = 4/3 slope_map = { (1,1):2, (4,3):1 } max for anchor = 2 -> total on line = 3 Repeat for each anchor point... Key: normalize slope as (dy/gcd, dx/gcd) tuple dx=0 -> vertical line key=(0,1) track exact duplicates separately
Basic Solution (Python)
from math import gcd
from collections import defaultdict

class Solution:
    def maxPoints(self, pts):
        n = len(pts)
        if n <= 2: return n
        res = 2
        for i in range(n):
            slopes = defaultdict(int)
            for j in range(i+1, n):
                dx = pts[j][0]-pts[i][0]
                dy = pts[j][1]-pts[i][1]
                g = gcd(abs(dx),abs(dy)) if dx else 1
                k = (0,1) if dx==0 else (dy//g,dx//g if dx>0 else -dy//g,-dx//g)
                slopes[k] += 1
            res = max(res, max(slopes.values())+1)
        return res
Optimized Solution (with Duplicate Handling)
from math import gcd
from collections import defaultdict

class Solution:
    def maxPoints(self, pts):
        n = len(pts)
        if n <= 2: return n
        res = 2
        for i in range(n-1):
            slopes, dups = defaultdict(int), 1
            for j in range(i+1,n):
                dx=pts[j][0]-pts[i][0]; dy=pts[j][1]-pts[i][1]
                if dx==0 and dy==0: dups+=1; continue
                g=gcd(abs(dx),abs(dy))
                k=(0,1) if dx==0 else ((dy//g,dx//g) if dx>0 else (-dy//g,-dx//g))
                slopes[k]+=1
            res=max(res,max(slopes.values(),default=0)+dups)
        return res
